## Short Description (132 characters)
Learn languages while browsing! AI-powered vocabulary annotation with SRS spaced repetition.

## Full Description

**TransLens** is a smart language learning assistant that helps you build vocabulary naturally while browsing the web.

### 🌟 Key Features

**📚 Learn in Context**
- Automatically detects foreign language text on web pages
- Annotates unfamiliar words with translations inline
- Supports Chinese, Japanese, Korean, French, German, Spanish, Russian, Arabic and more

**🧠 Spaced Repetition (SRS)**
- Built-in SM-2 algorithm (same as Anki)
- Reviews words just before you forget them
- Tracks mastery progress for each word

**🤖 AI-Powered Translation**
- Choose from OpenAI GPT-4o, Anthropic Claude, or any OpenAI-compatible API
- Context-aware translations based on sentence meaning
- Custom endpoint support for local models (Ollama, llama.cpp)

**⚙️ Fully Customizable**
- Select source and target languages
- Adjust word selection ratio (how many words to annotate)
- Set daily new word limits
- Disable on specific websites

**🔒 Privacy First**
- All data stored locally in your browser
- No accounts, no tracking, no analytics
- Your API key never leaves your device

### 🚀 How to Use

1. Install TransLens from Chrome Web Store
2. Click the extension icon to open Settings
3. Enter your OpenAI/Anthropic API key or configure a custom endpoint
4. Select the languages you want to learn
5. Browse any foreign language website – TransLens will do the rest!

### 📝 Supported Languages

**Source Languages:** Chinese (Simplified/Traditional), Japanese, Korean, French, German, Spanish, Russian, Arabic, or custom regex
**Target Languages:** English, Chinese, Japanese, Korean, French, German, Spanish

### 🔧 Technical Notes

- Requires your own API key (OpenAI, Anthropic, or compatible)
- Works with local AI servers (Ollama, llama.cpp, etc.)
- All settings and learning data stored in chrome.storage.local
